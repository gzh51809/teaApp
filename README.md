## 仿好茶链APP

react移动端项目

### 作者：黄子岚

### 项目架构

>开发IDE：Visual Studio

>前端开发：React + Redux + Ant Design + webpack + jsx，兼容Chrome，使用rem、flex、百分比布局

>后端开发：NodeJS(11.1.0) + mongoDB(4.0.4)
